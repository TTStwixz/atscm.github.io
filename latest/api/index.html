# Atviseproject

An atvise-scm project's configuration.

Hide members

The atvise-server's host. Defaults to localhost.

The atvise-server ports to use.

The login to use. Return false if no login is required (default).

The transformers to use in this project. Defaults to a single {@link DisplayTransformer}.

The atvise-server nodes that atvise-scm should sync. Defaults to the nodes AGENT, SYSTEM, ObjectTypes.PROJECT and VariableTypes.PROJECT.

An array of server related node ids. They should be ignored in a atvise-scm project as they are read-only.
static get ignoreNodes

These nodes (and their subnodes, if any) will be ignored by atvise-scm. Defaults to {@link Atviseproject.EditorRelatedNodes} combined with {@link Atviseproject.ServerRelatedNodes}.

# PartialTransformer

A transformer that transforms only some of the files read.

Hide members

constructor(options)

Creates a new partial transformer with the specified options.
filter

The filter stream used.
shouldBeTransformed(file)

true if file should be transformed.
applyToStream(stream, direction)

Applies the transformer to the given stream. It does so by running the following steps:

  • Pipe the {@link PartialTransformer#filter} stream
  • If PartialTransformer#applyToFilteredStream is overrridden, apply it.
  • Otherwise pipe {@link Transformer#withDirection}.
  • Restore the filter stream.
applyToFilteredStream(stream, direction)

Applies a stream transformer to the given, already filtered stream. Override this method if you want to pipe streams directly. Returning a falsy value (false, null, undefined, ...) invokes {@link Transformer#transformFromDB} or {@link Transformer#transformFromFilesystem} instead.

# NewlinesTransformer

A transformer that handles newline characters in files. During a pull, all breaks are converted the OS-native EOL character and a trailing newline is added (for better git diffs). On push, CRLF characters are used and those trailing newlines are removed again.

Hide members

transformFromFilesystem(file, enc, callback)

Removes trailing newlines and converts all breaks to CRLF.
get transformsReferenceConfigFiles

true because we want to transform all files.
